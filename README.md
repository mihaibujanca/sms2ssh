sms2ssh
=======

Ssh into a machine by sms.

It is just as simple as sending an sms to +44 113 320 2312 with the following to login:
{"username":"your_username",
            "host":"your_host",
            "port":"your_port",
            "password":"your_password",
            "encr":false"}
Then you can send comands: 
cd MYDIR
cd MYSUBDIR
wisecow
